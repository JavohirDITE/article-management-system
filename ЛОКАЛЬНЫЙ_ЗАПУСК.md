# 🚀 Локальный запуск / Local Setup

## Вариант 1: Встроенный PHP сервер (ПРОСТОЙ)

### Шаг 1: Скачай проект
```bash
git clone https://github.com/JavohirDITE/article-management-system.git
cd article-management-system
```

### Шаг 2: Установи зависимости SQLite (для локальной разработки)
```bash
# Проверь, что SQLite включен
php -m | grep -i sqlite
```

### Шаг 3: Создай конфигурацию для SQLite
Создай файл `config/database_sqlite.php`:
```php
<?php
return [
    'driver' => 'sqlite',
    'database' => __DIR__ . '/../database/articles.db',
    'options' => [
        PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION,
        PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
        PDO::ATTR_EMULATE_PREPARES => false,
    ]
];
```

### Шаг 4: Измени Database.php для автодетекта
В `classes/Database.php` добавь проверку на SQLite:
```php
public static function getConnection(): PDO
{
    if (self::$connection === null) {
        // Try SQLite first for local development
        $sqliteConfig = __DIR__ . '/../config/database_sqlite.php';
        if (file_exists($sqliteConfig)) {
            $config = require $sqliteConfig;
            $dbFile = $config['database'];
            
            if (!file_exists($dbFile)) {
                // Create database
                $dbDir = dirname($dbFile);
                if (!is_dir($dbDir)) mkdir($dbDir, 0777, true);
                
                self::$connection = new PDO('sqlite:' . $dbFile, null, null, $config['options']);
                self::setupSQLite();
                return self::$connection;
            }
            
            self::$connection = new PDO('sqlite:' . $dbFile, null, null, $config['options']);
            return self::$connection;
        }
        
        // Fallback to MySQL
        $config = require __DIR__ . '/../config/database.php';
        // ... остальной код
    }
}

private static function setupSQLite(): void
{
    $sql = file_get_contents(__DIR__ . '/../database/schema_fixed.sql');
    // Adapt for SQLite (replace AUTO_INCREMENT with AUTOINCREMENT, etc)
    $sql = str_replace('AUTO_INCREMENT', 'AUTOINCREMENT', $sql);
    $sql = str_replace('INT ', 'INTEGER ', $sql);
    self::$connection->exec($sql);
}
```

### Шаг 5: Запусти сервер
```bash
cd public
php -S localhost:8000
```

### Шаг 6: Открой браузер
```
http://localhost:8000
```

## Вариант 2: XAMPP (для MySQL)

### Шаг 1: Установи XAMPP
Скачай с: https://www.apachefriends.org/

### Шаг 2: Скопируй проект
```
C:\xampp\htdocs\article-manager\
```

### Шаг 3: Создай базу данных
1. Запусти XAMPP → Start Apache и MySQL
2. Открой: http://localhost/phpmyadmin
3. Создай БД: `articles_db`
4. Импортируй: `database/schema.sql`

### Шаг 4: Настрой config/database.php
```php
return [
    'host' => 'localhost',
    'dbname' => 'articles_db',
    'username' => 'root',
    'password' => '',
    'charset' => 'utf8mb4',
    // ...
];
```

### Шаг 5: Открой сайт
```
http://localhost/article-manager/public/
```

## Вариант 3: Docker (ПРОФЕССИОНАЛЬНЫЙ)

### Создай docker-compose.yml:
```yaml
version: '3.8'
services:
  web:
    image: php:8.2-apache
    ports:
      - "8000:80"
    volumes:
      - ./public:/var/www/html
      - ./classes:/var/www/html/classes
      - ./config:/var/www/html/config
      - ./views:/var/www/html/views
    depends_on:
      - db
      
  db:
    image: mysql:8.0
    environment:
      MYSQL_DATABASE: articles_db
      MYSQL_ROOT_PASSWORD: root
    ports:
      - "3306:3306"
    volumes:
      - db_data:/var/lib/mysql
      - ./database/schema.sql:/docker-entrypoint-initdb.d/schema.sql

volumes:
  db_data:
```

### Запусти:
```bash
docker-compose up -d
```

### Открой:
```
http://localhost:8000
```

## ⚠️ ВАЖНО для InfinityFree

На InfinityFree структура другая:
```
htdocs/
├── index.php      ← из public/
├── .htaccess      ← из public/
├── assets/        ← из public/assets/
├── classes/
├── config/
├── views/
└── database/
```

Все файлы из `public/` копируются в корень `htdocs`.

## 🐛 Troubleshooting

### Ошибка: "Database connection failed"
- Проверь `config/database.php`
- Убедись, что MySQL запущен
- Проверь пароль

### Ошибка: "Class not found"
- Проверь пути в `index.php`
- Должны быть: `__DIR__ . '/classes/...'`
- Не `__DIR__ . '/../classes/...'`

### SQLite не работает
```bash
# Windows
# Включи в php.ini:
extension=pdo_sqlite
extension=sqlite3
```
